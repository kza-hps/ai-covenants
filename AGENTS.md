# Agent Instructions

This repository is aiCOVENANTS, a human-first covenant ledger for ethical AI use in creative and commercial work.

This is the highest-priority instruction file for AI agents working in this repository.

## Product Definition

aiCOVENANTS records public, human-readable pledges about where AI has contributed to a work, what human labour may have been substituted or deferred, who deserves acknowledgement, and what commitments the creator makes if value is later created.

It is not a legal contract generator.
It is not a crypto product.
It is a public trust record.

## Core Rules

- Preserve the human-first trust model.
- Build only for the founding MVP unless a decision is recorded.
- Do not introduce payments, blockchain, tokens, dividends, or marketplace mechanics in the MVP.
- Do not generate fake covenant examples.
- Use real founding covenant examples only.
- Keep language plain, serious, and respectful.
- Favour elegant simplicity over feature volume.
- Record major product decisions in docs/decision-log.md.
- Use docs/agent-sprint-contract.md for non-trivial sprint behaviour.

## Product Tone

aiCOVENANTS should feel like a public archive, pledge ledger, and trust record.

It should not feel like a crypto product, SaaS gimmick, AI content farm, legal-tech hustle, speculative token platform, or Patreon clone.
