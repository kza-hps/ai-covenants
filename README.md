# aiCOVENANTS

aiCOVENANTS is a human-first pledge platform for declaring, publishing, and tracking ethical commitments around the use of AI in creative and commercial work.

It begins as a public covenant ledger for works created by Kauri Tukere under Hero's Journey Creative, including VouchMeApp, FASCIA, Ho and the Baby Eater, and aiCOVENANTS itself.

## Purpose

AI-assisted creation should be transparent, accountable, and human-centred.

aiCOVENANTS records where AI has contributed to a work, what human labour may have been substituted or deferred, who deserves acknowledgement, and what commitments the creator makes if value is later created.

It is not a legal contract generator. It is a public trust record.

## MVP Scope

The first version exists to create real, high-quality covenant records for a small founding catalogue:

- VouchMeApp
- FASCIA
- Ho and the Baby Eater
- aiCOVENANTS
- ElevenLabs personal voice-use covenant request

## Core Principles

- Human-first
- Public by default
- Plain language before legal language
- Attribution before automation
- Commitments before monetisation
- Edit history preserved
- Withdrawal recorded, not erased

## Non-goals

- Not a generic blockchain product
- Not a Patreon replacement
- Not a legal advice tool
- Not an AI marketplace
- Not a speculative token platform
- Not a platform for mass-generated pledges
